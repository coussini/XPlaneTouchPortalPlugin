To find and explore datarefs and commands in X-Plane, use DaterefTool. A dataref can uniquely identifies one value OR an array of values. Inside DatarefTool, when you click on a dataref, and press the "copy name" button, you save this dataref name into the clipboard.

For Example: sim/aircraft/view/acf_author represent one value AND AirbusFBW/ADIRUSwitchArray[1] represent an array of values and the value is at index 1 ->"[1]".

Knowing that an array of values always starts with index 0, index 1 represents the second value of this array. So, in this example, you have to add the string "[1]" after the dataref name. This is essential, because without the string "[1]", the "x-plane plugin for touch portal" will not be able to correctly use this variable.
For example AirbusFBW/ADIRUSwitchArray[1] or sim/aircraft/view/acf_author
Use a comment to list the values used by this dataref
